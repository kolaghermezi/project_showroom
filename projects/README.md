\# Ali Abrowshan — Engineering Project Showroom



This repository contains selected engineering projects in RF systems, software-defined instrumentation, embedded systems, biomedical signal processing, FPGA/HLS preparation, AI hardware/software acceleration, and computational electromagnetics.



The purpose of this repository is to present projects that demonstrate system-level engineering judgment: taking a technical problem, decomposing it into architecture, implementation, validation, and performance analysis.



\## Featured Projects



| Project | Area | Summary |

|---|---|---|

| Software-Based Spectrum Analyzer | RF / DSP / C++ | Desktop spectrum-analysis tool using C++, OpenGL, ImGui/ImPlot, FFT processing, Welch estimation, and binary I/O. |

| RF Cartography / RF Mapping System | RF Systems / SDR / Localization | Project direction focused on mapping RF environments using SDR, signal processing, spatial data, and hardware-aware sensing. |

| ECG Signal Acquisition System | Biomedical DSP | ECG signal-processing chain using MIT-BIH data, FIR filtering, FFT analysis, and ADC modeling. |

| Tensor Operator Kernel Suite | AI Hardware / Software | C++/Python/CUDA tensor operators relevant to NPU kernel and operator implementation. |

| Fixed-Point Neural Simulation | FPGA / HLS | Hardware-oriented fixed-point neural simulation with static memory structures, fixed-point arithmetic, and HLS preparation. |

| GPS + LoRa Vehicle Tracking System | Embedded / RF / IoT | Embedded tracking system using GPS and LoRa communication for long-range vehicle telemetry. |

| Load Slammer Test Platform | Power Electronics / Embedded | Pulsed electronic load platform using microcontroller control, MOSFET switching, temperature sensing, and UI feedback. |

| Spatial Transcriptomics Registration Workflow | Computational Biology / Imaging | Workflow exploration using transcriptomic spatial data, AllenSDK, and image registration methods. |
